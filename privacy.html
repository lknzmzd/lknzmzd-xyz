SIG-PRIVACY / V2.5

Signals Privacy

The V2.5 subscription layer stores only the minimum information required to operate LKNZMZD Signals.

When the Cloudflare Worker backend is connected, the system stores email address, selected interests, subscription status, source page, consent version, timestamps, and optional non-reversible IP hash if configured. It does not need to store raw IP addresses.

Subscriber data is stored in Supabase and is not exposed through public browser keys. Public writes are disabled; the Cloudflare Worker writes using a server-side service role key.

To unsubscribe or request removal before a self-service unsubscribe endpoint exists, email ilkinazimzade@lknzmzd.com.
